\begin{abstract}
	Phase 1 implementation includes the following steps:
	\begin{itemize}
		\item Custom k8s fully automated deployment on Azure:
		\begin{itemize}
			\item Deploy Private Virtual Network with Private Virtual Subnets
			\item Deploy Linux Virtual Machines (VM) (LB, k8s nodes)
			\item Configure High Availability (HA) Load Balancer (LB)
			\item Start Prime Master k8s node
			\item Once Prime Master k8s node reach Ready state deploy rest of nodes (Masters, Workers)
		\end{itemize}
		\item Custom k8s fully automated deplpoyment on AWS (TODO)
		\item Custom k8s fully automated deplpoyment on Google (TODO)
	  \end{itemize}
\end{abstract}
